# clinic-admin-lite (P18) — Demo

Clinic admin lite: appointments, patient ledger, Rx print, WhatsApp follow-ups for small practices

## Run

```bash
cd clinic-admin-lite
node demo/run-demo.mjs
node src/cli.js --json fixtures/sample.json
npm test
```

Patients + appts + basic billing + WA reminder
