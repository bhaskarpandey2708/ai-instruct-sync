# climate-ops-meter

> P25 · Climate / ESG

**Lightweight Scope 1–3 carbon estimate and supplier survey tool for mid-market exporters**

## Status

Offline **MVP core** (v0.1.0-beta.0) — zero runtime dependencies, litmus-tested, demo-ready.
Part of the Bhaskar Pandey product portfolio under `Documents/development`.

## Quick start

```bash
cd climate-ops-meter
npm test
npm run demo
node src/cli.js --help
```

Activity data forms + emission factors + PDF report
