Your Claude skill is fixed.
Your teammate’s Cursor skill is still broken.
Nobody notices until the same bug ships twice.

That isn’t a model failure.
It’s a packaging failure.

We wouldn’t ship npm without versions.
But the skills that steer AI coding agents still travel as:
“final.md” · Slack snippets · Drive copies · hope.

I built skill-sync — a free, local, open-source alpha CLI that treats agent skills more like packages:

→ Validate name, version, and skill entries  
→ Diff local vs remote: added vs updated  
→ JSON for scripts and CI  
→ Zero runtime dependencies — nothing leaves your machine  

Try it:

npx @bhaskarauthor/skill-sync

Honest limit: offline core (validate + install plan), not a hosted private registry. Discipline first; platform later.

If your “source of truth” is a shared folder and goodwill, this is the boring layer underneath. Feedback welcome.
