# appt-book-india (P17) — Demo

Lightweight appointment booking with WhatsApp reminders for salons, tutors, clinics lite

## Run

```bash
cd appt-book-india
node demo/run-demo.mjs
node src/cli.js --json fixtures/sample.json
npm test
```

## MVP

Public booking page + staff calendar + WA reminder via Cloud API

## Platforms

Web,Android,API

## Commercial path (later)

OSS core stays free; paid layer = team dashboard / hosted / compliance export.
