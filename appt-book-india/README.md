# appt-book-india

> P17 · India SMB / Appointments

**Lightweight appointment booking with WhatsApp reminders for salons, tutors, clinics lite**

## Status

Offline **MVP core** (v0.1.0-beta.0) — zero runtime dependencies, litmus-tested, demo-ready.
Part of the Bhaskar Pandey product portfolio under `Documents/development`.

## Quick start

```bash
cd appt-book-india
npm test
npm run demo
node src/cli.js --help
```

Public booking page + staff calendar + WA reminder via Cloud API
