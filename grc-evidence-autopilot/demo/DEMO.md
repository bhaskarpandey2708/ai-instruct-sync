# grc-evidence-autopilot (P14) — Demo

Auto-collect compliance evidence (access logs, policies, screenshots) for SOC2/ISO startups

## Run

```bash
cd grc-evidence-autopilot
node demo/run-demo.mjs
node src/cli.js --json fixtures/sample.json
npm test
```
