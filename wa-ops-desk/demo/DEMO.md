# wa-ops-desk (P15) — Demo

WhatsApp-first ops desk for Indian SMBs: orders, FAQs, staff routing, CRM lite

## Run

```bash
cd wa-ops-desk
node demo/run-demo.mjs
node src/cli.js --json fixtures/sample.json
npm test
```

Shared inbox + labels + quick replies + 1 catalog flow
