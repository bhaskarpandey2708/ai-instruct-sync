# care-companion (P22) — Demo

Family care coordination app: meds, appointments, shared notes for elders

## Run

```bash
cd care-companion
node demo/run-demo.mjs
node src/cli.js --json fixtures/sample.json
npm test
```

## MVP

Shared list + med reminders + appt calendar for 1 family

## Platforms

iOS,Android,Web
